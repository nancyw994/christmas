## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the displayed URL (usually `http://localhost:3000`)

## How to Play

1. Enter the secret password to access the application
2. Click "Start decorating" to begin
3. Drag ornaments from the tray to the correct positions on the tree
4. Use the crossword puzzle clues to find the correct placement
5. Once all ornaments are placed correctly, click the glowing star to read a special message
